# App Store Privacy Nutrition Label — Stashbar

## Summary

Stashbar does not collect any user data. All functionality is local to the device.
The app does not use analytics, advertising, crash reporting, or any third-party services that transmit data.

## Notes for Reviewer

- The app uses App Sandbox (`com.apple.security.app-sandbox`) and does not request any network entitlements.
- All thumbnail caching is performed locally on disk within the app's container.
- Folder access is managed via security-scoped bookmarks created from explicit NSOpenPanel user selections.
- No user content leaves the device.
