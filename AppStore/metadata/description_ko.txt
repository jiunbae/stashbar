Stashbar은 자주 확인하는 폴더를 메뉴바에서 빠르게 열어 최신 파일을 확인할 수 있게 해주는 macOS 생산성 유틸리티입니다.

스크린샷, 다운로드, 작업 폴더처럼 반복해서 여는 위치를 메뉴바에 고정하고, 최근 항목을 한곳에서 정리된 상태로 보여줍니다. Finder를 계속 앞에 두지 않아도 필요한 파일을 바로 찾고 열 수 있습니다.

주요 기능
• 여러 폴더 동시 감시: 원하는 폴더를 추가하면 실시간으로 변경 사항을 반영합니다
• 3가지 보기 모드: 아이콘, 목록, 계층 보기를 지원하며 상황에 맞게 전환할 수 있습니다
• 빠른 미리보기: Quick Look을 지원하여 Space 키로 파일을 즉시 확인할 수 있습니다
• Finder 단축키: 복사(⌘C), 잘라내기(⌘X), 붙여넣기(⌘V), 휴지통 이동(⌘⌫)을 그대로 사용합니다
• 드래그 앤 드롭: 다른 앱이나 Finder로 파일을 바로 드래그할 수 있습니다
• 디스크 썸네일 캐시: 앱을 재실행해도 미리보기가 즉시 표시됩니다
• 로그인 시 자동 실행: 항상 메뉴바에서 대기하며 SMAppService로 안정적으로 동작합니다
• App Sandbox 호환: security-scoped bookmark로 선택한 폴더 접근 권한을 안전하게 영속화합니다

반복적으로 파일을 열고 확인하는 흐름을 더 짧고 자연스럽게 만드는 것에 초점을 맞춘 메뉴바 전용 앱입니다.
