Stashbar은 자주 확인하는 폴더를 메뉴바에 고정해 최근 파일을 빠르게 찾고 여는 macOS 유틸리티입니다.

스크린샷, 다운로드, 작업 폴더처럼 반복해서 확인하는 위치를 한곳에 모아 보여줍니다. Finder 창을 먼저 열 필요 없이 메뉴바 팝오버에서 최신 항목을 확인하고, Quick Look으로 미리 보고, 필요한 작업을 바로 이어갈 수 있습니다.

주요 기능
• 여러 폴더를 동시에 고정하고 빠르게 전환
• 아이콘, 목록, 계층 보기와 정렬 옵션 지원
• Quick Look 미리보기와 Finder 스타일 키보드 단축키 지원
• 드래그 앤 드롭, 복사/붙여넣기, 휴지통 이동 지원
• 디스크 캐시와 보안 북마크로 재실행 후에도 빠른 접근 유지
• 로그인 시 자동 실행으로 항상 메뉴바에서 대기

Stashbar은 반복적으로 파일을 찾고 확인하는 흐름을 더 짧고 자연스럽게 만드는 메뉴바 전용 도구입니다.
