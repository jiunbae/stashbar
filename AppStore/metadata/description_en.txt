File Stack is a macOS productivity utility that puts your frequently accessed folders in the menu bar so you can check recent files instantly.

Pin folders like Screenshots, Downloads, or project directories to the menu bar and browse their latest contents without keeping Finder in front. File Stack gives you a compact, organized view of recent files exactly where you need them.

Features
• Watch multiple folders: Add folders and see live updates as files change
• Three view modes: Icon, list, and hierarchy views with a size slider
• Quick Look preview: Press Space to instantly preview any selected file
• Finder-style shortcuts: Copy (⌘C), cut (⌘X), paste (⌘V), and move to trash (⌘⌫)
• Drag and drop: Drag files directly to other apps or Finder windows
• Disk thumbnail cache: Thumbnails persist across launches for instant browsing
• Launch at login: Always available in the menu bar via modern SMAppService
• App Sandbox ready: Uses security-scoped bookmarks to safely persist folder access

Designed from the ground up as a menu bar app to make your repeated file-checking workflow shorter and more natural.
